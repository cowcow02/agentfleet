Daemon Setup

Install and connect from any machine:

1. Get your token from the Members table above
2. Configure daemon: edit ~/.agentfleet/agents.yaml
hub:
token: <your-token>
3. Start the daemon:
$ node daemon/index.js

The daemon will automatically register your agents with this hub and start accepting dispatches.
